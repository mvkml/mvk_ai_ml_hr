# ADR001 - Use FastAPI as Backend Framework

## Status: Accepted

## Context
We need a backend API framework for the mvkhr HR system.

## Decision
We will use **FastAPI** (Python) as our backend framework.

## Reasons
- High performance (async support)
- Auto-generates API docs (Swagger/OpenAPI)
- Strong Python ecosystem for ML/AI agents
- Easy integration with Claude AI agents

## Consequences
- Team needs Python knowledge
- All backend services will be Python-based
